# Annotation figures

Drop teaching screenshots here and embed them in the task guides, e.g.

```markdown
![ribs example](figs/ribs_annotated_example.png)
```

**Best figures = your own annotated CT slices** — they're yours to use freely and
are exactly on-task (show the real palette + a real case).

**Using a Radiopaedia figure?** Those are CC BY-NC-SA. For non-commercial teaching
that's fine if you (a) **download** the image into this folder rather than hotlink
their CDN (hotlinks rot and are discouraged), (b) **credit** the case author and
link the case, and (c) keep the **same licence**. Otherwise just **link** to the
Radiopaedia article (always fine, always renders) — which is what the guides do.
